What I've Learned during Intro to Programming

A collection of tags I learned during class:

  • <p>...</p> Paragraph tag, makes a block of text that is very common on any website.

  • <ul>...</ul> Unordered List, specifies an unordered list as opposed to a ordered list.

  • <ol>...</ol> Ordered List, specifies an ordered list as opposed to an unordered list.

  • <li>...</li> List Item, creates a bullet point or numbered item in a unordered or ordered list.

  • <h1>...</h1> h1-6 Headings, Creates a heading that is styled to be bold and larger than normal text. The lower the number after the h the less important the tag.

  • <div>...</div> Divisions for grouping, very commonly used to group parts of code and make it easier to add properties no many things at once.

  • <br>...</br> Break, a break in the text adding more space inbetween two elements. Multiple breaks can be used in conjunction to add more space.

  • <strong>...</strong> Makes text bold by wrapping a part of text within the strong tag and inserting it into a paragraph or list item (<p> and <li>).

  • <img>...</img> Images, adds an image on the webpage, often used with the src element to specify where the image comes from.

  • <body>...</body> The content of the website, tells the browser that it has encountered the area where the content for the website is.

  • <html>...</html> Makes the browser read the page as html, tells the browser that this website is written in html and not a different language.

  • <title>...</title> Sets the title of the website, specifies what the browser should call the website. The title appears at the top of the tab in the browser.

  • <alt>...</alt> Sets alternate text for an image, if the computer is reading the page it will automatically read the alt text instead. This is used for instance if someone blind wants to use your page.

  • <a>...</a> For creating Links, this creates links for your page that lead to other pages. This tag is also used to make a navbar.

  • <em>...</em> Makes text italic, makes text italic by wrapping the text within the tag and inserting it into a paragraph or list item(<p> and <li>).

  • <!...text here...> Makes a comment inside the code, this is invisible to the viewer unless they inspect the code.

  • <style>...</style> Tells the browser how to style the content of the page, this is used to tell the browser what part of the website is css.

  • <table>...</table> Specifies what code is part of the table, has all of the tables content inside.

  • <thead>...</thead> Table head code, specifies that this part of the table is the header. Followed by the table header tag (<th>).

  • <tbody>...</tbody> Table body code, specifies that this part of the table is the body and contains the main data. Followed by the table data tag(<td>).

  • <tr>...</tr> Table row, one line or row of the table. Tells the browser that it should create a new row.

  • <th>...</th> Table header, contains the data for a singular table head cell.

  • <th>...</th> Table data, contains the data for a singular table body cell.
